# back-teady-study
## 스터디 규칙
1. 매주 각자 미리 정한 목표 달성
2. 각자 폴더에 학습한 내용 마크다운으로 작성
3. 늦제출 : 1000원💸
4. 미제출 : 2000원💸💸
5. 매주 `일요일` 자정까지 출석으로 인정
6. 다음주 인증일까지 미제출분 제출 시 늦제출, 그 때까지도 안 하면 미제출
